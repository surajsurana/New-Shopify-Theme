3 — Footer Redesign

Copy fix: "Mumbai" / worldwide-shipping messaging previously appeared three times (logo sub-line, address block, bottom service line) plus a fourth near-duplicate ("Ships worldwide" locale line). Now: the logo sub-line's "Mumbai" stays as brand identity (heritage marker, like Sabyasachi's "Kolkata"), and the Contact column's address becomes the single authoritative logistics line, with the DHL Express detail folded in rather than repeated separately. The bottom bar's redundant service/locale lines are removed outright, not relocated — it now carries only copyright + legal links, the calm luxury-standard bottom bar.

Mobile fix: resize below using the device buttons. Watch Contact + Inner Circle pair correctly at tablet width (no orphaned empty cell, the old bug), then watch the small-phone view move to two columns — Revision v2: Suraj approved the footer design overall but asked for a two-column phone layout to cut scroll length (was previously one hairline-divided column all the way down). Now: Brand+Social pairs with Collections, Maison pairs with Contact (so "Book an Appointment" sits directly beside the phone/WhatsApp line), and Inner Circle stays full-width alone since it holds a real email form. Every block carries an explicit grid-column and grid-row at this width — none rely on auto-flow — specifically so the tablet tier's orphaned-cell bug can't recur here. YouTube added to the social row; Press removed from the Maison column (no live page exists behind it yet).

Revision v3 (divider cleanup): v2's phone layout paired its two columns with a horizontal rule under each row PLUS a vertical rule splitting each row into columns — 5 rule lines stacked in one footer. Suraj: too many dividers, reads as a spreadsheet grid. Fixed: all internal dividers removed — both the row-dividers and the column hairline — relying purely on the existing padding (already generous) to separate the three stacked blocks, the same way the desktop 5-column footer above separates its columns with gap alone and zero rules. Only one hairline survives, below the newsletter block — that's not a new divider, it's the same closing line desktop already has before its copyright bar, kept for cross-breakpoint consistency. Resize to Mobile below to see it live.
